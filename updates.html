Changelog

    Date: 17/06/2021

  • Added a Healing column to the tracker. As you may be able to guess, numbers entered in this column are added to the health of the respective creature.
  • Made some changes to page scaling at different browser and windows sizes. The table should now be more readable and make more efficient use of space at smaller sizes.
  • Added a more robust Undo/Redo system. You can now undo changes made in the tracker fields, including damage and healing, by using the buttons in the top left or using keyboard shortcuts. This system may not work properly with loaded data that was created before this update.
  • Adjusted the sizes of the colour coding options to make them more useable on smaller devices.
  • Made some minor code optimizations (will likely only be noticeable when using many rows in the tracker).
  • Fixed some toolbar display elements not changing text colour correctly.
  • Help info will now appear as a modal covering the screen, rather than a popover that was difficult to read on smaller window sizes.
  • The page should now correctly ask for confirmation when refreshing, closing or leaving the page if any data has been changed in the current session.

    • Date: 06/07/2020

  • Added this page.
  • Added a new toolbar to the top of the screen and removed the Settings button. Options from the Settings menu have been moved to the toolbar.
  • Fixed an issue when importing a saved file that was lacking initiative counts for some rows. All rows with blank initiative cells should now load containing a placeholder 0, rather than not loading at all.
  • Changed the text colour in yellow rows to make it more readable.
  • Changed the default sorting order of the table to place raw initiative numbers first. The sorting order can still be changed in the settings.
  • Fixed some browser compatibility issues.
